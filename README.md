# 💼 HireHub – Role-Based Job Portal

HireHub is a full-stack MERN application that allows Job Seekers and Recruiters to interact on a single platform.

## 🚀 Features

### 👤 Job Seeker
- Signup/Login authentication (JWT)
- View available jobs
- Apply for jobs
- Track applied jobs

### 🏢 Recruiter
- Post new jobs
- View applicants
- Manage job listings

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Styling: CSS

## 🔐 Authentication

Role-based authentication system:
- Protected routes
- Secure API access
